# 项目需求

## 初始需求

请帮我编写两个 Python 脚本，用于抓取指定小说网站的内容

## 扩展需求

需要完成一个自动化爬虫管道，用于抓取、下载和合并中文网络小说：

### 目标
- 爬取 zhswx.com 上的小说《五仙门》（2883章）
- 将所有章节下载保存为本地文件
- 将下载的章节合并成分册的电子书文件（每册500章左右）

### 功能要求

1. **目录解析脚本**
   - 抓取小说目录页面，提取所有章节的元数据（章节号、标题、链接）
   - 保存为 JSON 格式，便于后续处理

2. **内容下载脚本**
   - 支持增量下载（断点续跑）
   - 针对目标网站的 HTML 结构进行正文提取
   - 自动清理广告和无关内容
   - 实时更新下载状态，便于进度跟踪

3. **分册合并脚本**
   - 按章节编号将下载的章节文件合并成分册
   - 保持正确的章节顺序和格式
   - 支持缺失文件的优雅处理

### 技术要求

- **语言**：Python 3，尽量使用标准库
- **外部依赖**：requests（HTTP 请求）、beautifulsoup4（HTML 解析）
- **编码**：UTF-8，支持中文
- **错误处理**：健壮的异常处理，支持重试和断点恢复
- **目标网站**：zhswx.com（具体 HTML 结构需要分析）

### 核心特性

- ✓ 完整的抓取、下载、合并流程
- ✓ 中断恢复能力（checkpoint 机制）
- ✓ 错误章节自动跳过
- ✓ 中文字符编码处理
- ✓ 实时进度更新
