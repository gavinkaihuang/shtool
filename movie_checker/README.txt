movie_checker / video_checker.py 说明

功能目的
- 扫描指定目录下的视频文件。
- 尝试识别损坏、不完整或异常视频文件。
- 将扫描结果输出为 CSV 报告，便于后续筛查和处理。

检测逻辑（当前版本）
- 先用 ffprobe 读取视频时长等基础信息：
	- 读取失败通常判为 损坏。
	- 时长为空判为 异常。
- 再用 ffmpeg 抽查文件尾部（最后 5 秒）：
	- 读取失败判为 不完整。
- 两步都通过则判为 健康。

前置依赖
- Python 3
- ffprobe（通常随 ffmpeg 一起安装）
- ffmpeg

可用以下命令检查依赖是否安装成功：
- ffprobe -version
- ffmpeg -version

使用方法
1. 进入脚本所在目录（可选）
	 cd movie_checker

2. 方式一：命令行传入扫描目录（推荐）
	 python3 video_checker.py /path/to/your/videos

3. 方式二：不传参数，运行后手动输入目录
	 python3 video_checker.py
	 然后按提示输入目录路径。

输出结果
- 默认会在当前运行目录生成报告文件：
	video_health_report.csv
- 报告列如下：
	- 文件名
	- 状态
	- 详情
	- 完整路径

状态说明
- 健康：基础信息可读取，尾部抽查通过。
- 损坏：文件头/索引读取失败。
- 异常：时长信息缺失。
- 不完整：尾部抽查失败，可能是截断或下载未完成。
- 超时：单文件检测超时。
- 错误：运行期异常（例如依赖缺失等）。

注意事项
- 当前方案是实用型快速筛查，不等同于逐帧全量解码校验。
- 某些封装格式或特殊编码可能出现误判。
- 如果扫描目录很大，耗时会明显增加。
